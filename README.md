# swe-study

A quick start in Python 3 and JavaScript (ES6).

Based on [gpdowning/cs373](https://github.com/gpdowning/cs373).

## Repo Structure
```
|- python - Python 3 basics
\--- functions
 |----- comprehension_and.py - basic list/set/dict comprehension and generators
 |----- copy.py - 'copy' and 'deepcopy'
 |----- equality.py - '==' and 'is'
 |----- exceptions.py - exception raising and handling
 |----- for_loop.py - for loops
 |----- iterator.py - basic iteration
 |----- map_and_filter.py - map and filter functions
 |----- operators_and_syntax.py - basic Python operators
 |----- plus_equals.py - in-depth '+=' v. '__ + __'
 |----- range-count-zip.py - range, count, and zip functions
 |----- yield.py - yield operator
\--- structures
 |----- add_properties.py - add a constructor; make a class iterable, indexable
 |----- classes_and_methods.py - basic class and method creation
 |----- dict.py - basic dictionary
 |----- list.py - basic list
 |----- set.py - basic set
 |----- tuple.py - basic tuple
 |----- types.py - Python types
\--- about.md - about Python
\--- concepts.md - larger programming concepts
\--- misc.py - miscellaneous Python notes
\--- run.py - instructions for running Python code
```
